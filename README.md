# 🎮 Videojuego 2D - Pacman Ghost Hunter 👻

## 📌 Contexto

En el presente proyecto se desarrolló un videojuego en 2D utilizando la API Canvas de HTML5, como parte de la materia de Graficación en la carrera de Ingeniería en Sistemas Computacionales. El objetivo principal es aplicar conceptos de programación gráfica, animación y manejo de eventos en aplicaciones web interactivas.

El videojuego está inspirado en la temática clásica de Pac-Man, adaptando su estilo visual a una mecánica diferente basada en interacción con el mouse.

---

## 🎯 Objetivo

Desarrollar un videojuego 2D funcional que implemente:
- Uso de Canvas para renderizado gráfico
- Movimiento dinámico de múltiples objetos
- Interacción con el usuario mediante eventos de mouse
- Animaciones y detección de colisiones
- Integración de elementos multimedia como imágenes y audio

---

## 💡 Justificación

El desarrollo de este proyecto permite reforzar conocimientos en:
- Programación en JavaScript
- Manipulación del Canvas en HTML5
- Lógica de animación y física básica (movimiento y colisiones)
- Diseño de interfaces interactivas

Además, este tipo de aplicaciones simulan escenarios reales en el desarrollo de videojuegos y aplicaciones gráficas, fortaleciendo habilidades prácticas del estudiante.

---

## ⚙️ Operación del videojuego

El videojuego consiste en eliminar fantasmas que se desplazan dentro del canvas.

### 🔹 Características principales:

- Se generan 25 objetos (fantasmas) en pantalla
- Los objetos se mueven en diferentes direcciones:
  - Horizontal
  - Vertical
  - Diagonal
  - Circular
- Cada objeto tiene velocidad distinta
- Los objetos rebotan al colisionar entre sí y con los bordes del canvas
- Al detectar una colisión, se muestra una animación de choque
- El usuario puede hacer clic sobre los objetos para eliminarlos
- Al eliminar un objeto, este reaparece en una posición aleatoria
- Se mantiene siempre un total de 25 objetos en pantalla
- Se incluye un contador de objetos eliminados
- Se integra música de fondo y efectos de sonido
- Se utiliza un cursor personalizado para mejorar la experiencia de usuario
